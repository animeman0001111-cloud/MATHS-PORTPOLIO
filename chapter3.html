Chapter 3

Solving Linear Equations

Gaussian Elimination · LU Factorization · Matrix Form · Engineering & AI Applications

Linear Equations

A linear equation has variables with power one, forming a straight line (in 2D) or a hyperplane (in nD) when graphed.

ax + by = c

A system of linear equations contains multiple equations that must all be satisfied simultaneously:

a₁x + b₁y = c₁ a₂x + b₂y = c₂

Matrix Form: Ax = b

Any linear system is expressible as a compact matrix equation:

A · x = b [ 2 3 ] [ x ] [ 6 ] [ 1 -1 ] [ y ] = [ 0 ]

A is the coefficient matrix, x is the variable vector, b is the constants vector. Solution: x = A⁻¹b when A is invertible.

Gaussian Elimination

Systematically converts the augmented matrix to row echelon form using elementary row operations, then solves by back-substitution.

1

Write [A|b] — the augmented matrix combining coefficients and constants.

2

Forward elimination — create zeros below each pivot using row ops.

3

Back substitution — solve each variable from the last equation upward.

Possible Outcomes

Unique Solution

Exactly one answer. Lines (or planes) meet at a single point.

No Solution

Inconsistent — parallel lines that never intersect.

Infinite Solutions

Lines coincide. Every point on the line is valid.

LU Factorization

Decomposes A into Lower triangular L and Upper triangular U, enabling fast multi-solve:

A = L × U Ly = b → forward substitution Ux = y → back substitution

Once L and U are computed, any number of right-hand sides b can be solved cheaply — essential in physics simulations and FEM.

Real World Applications

  • Engineering: Circuit analysis via Kirchhoff's laws; structural load calculations
  • Computer Graphics: Solving vertex positions in 3D rendering pipelines
  • AI & ML: Training linear regression via least squares
  • Economics: Leontief input-output models
  • Robotics: Inverse kinematics — computing joint angles for a target position
