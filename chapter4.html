Introduction to Vector Spaces

A vector space is a mathematical system where vectors can be added together and scaled by real numbers while staying within the same set. They model everything from physical forces to machine learning feature spaces.

Vector Space V requires: 1. u + v ∈ V (closure under addition) 2. c · v ∈ V (closure under scalar multiplication)

Axioms of a Vector Space

A set V is a vector space over ℝ when all 8 axioms hold:

Closure (+)

u + v ∈ V for all u, v ∈ V

Commutativity

u + v = v + u

Associativity

(u + v) + w = u + (v + w)

Zero Vector

∃ 0 such that v + 0 = v

Additive Inverse

∃ −v such that v + (−v) = 0

Closure (·)

c · v ∈ V for all c ∈ ℝ

Distributivity

c(u + v) = cu + cv

Identity

1 · v = v

Subspaces, Basis and Dimension

A subspace is a subset of a vector space that is itself a valid vector space. A basis is the minimal set of linearly independent vectors that spans the entire space. The number of basis vectors equals the dimension — the degrees of freedom.

dim(ℝ³) = 3 → 3 degrees of freedom Standard basis: {[1,0,0], [0,1,0], [0,0,1]}

Applications

🤖 Robotics

A 6-DOF robot arm lives in a 6-dimensional configuration space. Motion planning is navigation through this space.

🎨 RGB Colours

All visible colours form a 3D vector space. Basis vectors are pure red, green, and blue.

🧠 Machine Learning

PCA reduces high-dimensional feature spaces to the subspace capturing maximum variance.

💻 Compression

JPEG operates by projecting image data onto DCT basis functions — a change of basis in 2D vector space.
